║                                                                               ║
║  Where: SFM = Surface Feet per Minute                                         ║
║         Vc = Cutting speed (m/min)                                            ║
║         D = Diameter (inches or mm)                                           ║
║                                                                               ║
║  FEED RATE CALCULATION                                                        ║
║  ─────────────────────                                                        ║
║  Feed Rate (IPM) = RPM × IPR                      [Turning]                   ║
║  Feed Rate (IPM) = RPM × IPT × Number of Teeth    [Milling]                   ║
║  Feed Rate (mm/min) = RPM × mm/rev                [Turning]                   ║
║  Feed Rate (mm/min) = RPM × mm/tooth × Teeth      [Milling]                   ║
║                                                                               ║
║  CHIP THINNING (RADIAL ENGAGEMENT)                                            ║
║  ─────────────────────────────────                                            ║
║  Actual Chip = Programmed × √(ae/D)                                           ║
║                                                                               ║
║  Chip Thinning Compensation Factor:                                           ║
║  ┌──────────────┬────────────────┬────────────────────┐                       ║
║  │ ae/D (WOC)   │ Factor         │ Feed Increase      │                       ║
║  ├──────────────┼────────────────┼────────────────────┤                       ║
║  │ 100% (slotting)│ 1.00         │ No adjustment      │                       ║
║  │ 50%          │ 1.41           │ +41%               │                       ║
║  │ 25%          │ 2.00           │ +100%              │                       ║
║  │ 15%          │ 2.58           │ +158%              │                       ║
║  │ 10%          │ 3.16           │ +216%              │                       ║
║  │ 5%           │ 4.47           │ +347%              │                       ║
║  └──────────────┴────────────────┴────────────────────┘                       ║
║                                                                               ║
║  METAL REMOVAL RATE (MRR)                                                     ║
║  ────────────────────────                                                     ║
║  MRR (in³/min) = WOC × DOC × Feed Rate                                        ║
║  MRR (cm³/min) = ae × ap × Vf / 1000                                          ║
║                                                                               ║
║  Where: WOC = Width of Cut (ae)                                               ║
║         DOC = Depth of Cut (ap)                                               ║
║         Vf = Feed Rate                                                        ║
║                                                                               ║
╚═══════════════════════════════════════════════════════════════════════════════╝
```

### 9.3 Wear Patterns and Diagnosis

```
╔═══════════════════════════════════════════════════════════════════════════════╗
║  TOOL WEAR DIAGNOSIS                                                          ║
╠═══════════════════════════════════════════════════════════════════════════════╣
║                                                                               ║

║  ─────────────────┼──────────────────────────┼────────────────────────────    ║

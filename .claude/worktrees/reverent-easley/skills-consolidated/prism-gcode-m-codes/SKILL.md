# prism-gcode-m-codes
## Purpose
Reference for G-code M-codes (machine auxiliary functions) across CNC controller families.
## When to Use
- Decoding M-codes in G-code programs (M03, M08, M30, etc.)
- Generating G-code with correct M-codes for target controller
- Comparing M-code behavior across FANUC, HAAS, Siemens, Mazak
## Examples
- M03 S12000 — spindle on CW at 12000 RPM
- M08 — coolant on (flood)
- M30 — program end and reset
